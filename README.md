A simple web form app built with Vue.

Live demo: https://goofy-mestorf-0699ac.netlify.com

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
